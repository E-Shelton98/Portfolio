TMDB-Movies

An extension of a tutorial from 50 Projects in 50 Days, allows for searching of a movie or television show name and displays matches. Has a backend to hide the API key from being public. Hosted on Vercel.

This project is done entirely in vanilla Javascript, HTML, and CSS, besides the backend which is using Express and hosted on Heroku.

A project that allows for a person to assign tasks to there 'workers'. Designed to be used by supervisors, teachers, parents, or anyone who needs to simply and efficiently delegate tasks to people while keeping track of what is assigned and has been completed.
